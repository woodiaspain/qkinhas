# Qkinhas Landing Page

## Sobre el proyecto
Landing page para Qkinhas, la primera cafetería especializada en café y postres en lata de A Coruña. Fundada por Yonathan (@yonathantv), ubicada en Cuesta de la Unión 3, bajo izquierda, barrio de La Buena Vida.

## Stack
- HTML5 semántico + CSS3 + Vanilla JS (sin frameworks)
- Un único archivo index.html autocontenido
- Google Fonts vía CDN
- Sin dependencias npm

## Estética
- Estilo retro americano / vintage diner meets street food moderno
- Paleta: crema (#FFF8EE), marrón oscuro (#2C1A0E), naranja vibrante (#E8622A), dorado (#D4A017)
- Tipografía display: Playfair Display o similar serif dramático
- Tipografía body: DM Sans o similar sans-serif limpio
- Mascota protagonista: galleta con gorra y lata (cookie character)
- Sensación: cálido, artesanal, con personalidad urbana y community-driven

## Tono de copy
- Español, cercano, con humor y personalidad
- Enfocado en A Coruña / barrio La Buena Vida
- Destaca lo limitado, lo artesanal, lo instagrameable

## Reglas
- No placeholders. Todo el copy es real y final
- Mobile-first, responsive perfecto
- Animaciones CSS sutiles (no JS pesado)
- Performance: imágenes como emojis/SVG o CSS art donde no haya assets reales
- El logo es la mascota galleta (usar emoji 🍪 estilizado o describir como "cookie character con gorra")
